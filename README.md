# Медитация для фокуса

10-дневная программа медитационных практик для развития устойчивого внимания и осознанности.

## Описание

Этот лендинг предоставляет структурированный подход к медитации с акцентом на практики с открытыми глазами, помогая интегрировать навыки осознанности в повседневную жизнь.

### Основные возможности

✅ **10-дневная программа** с постепенным усложнением практик
✅ **Адаптивная система прогресса** — следующий день открывается после завершения предыдущего
✅ **Чеклисты для каждой практики** с сохранением прогресса в браузере
✅ **5 уникальных анимаций** для поощрения достижений (confetti, ripple, sparkle, pulse, check bounce)
✅ **Минималистичный дизайн** в психологическом стиле
✅ **Адаптивная верстка** для комфортного использования на всех устройствах

## Установка и запуск

### Требования

- Node.js версии 18 или выше
- npm (входит в комплект Node.js)

### Быстрый старт

1. **Установите зависимости:**
   ```bash
   npm install
   ```

2. **Запустите приложение:**
   ```bash
   npm start
   ```

   Приложение автоматически откроется в браузере по адресу `http://localhost:5173`

### Другие команды

- `npm run dev` — запустить dev-сервер без автоматического открытия браузера
- `npm run build` — собрать проект для продакшена
- `npm run preview` — просмотреть production-сборку локально

### Docker

Для запуска через Docker (требуется Docker):

```bash
npm run docker:up
```

Приложение будет доступно по адресу `http://localhost:3000`

Остановка:

```bash
npm run docker:down
```

## Как использовать

1. **Первый запуск:**
   - При первом открытии приложения появится модальное окно
   - Выберите дату начала вашей практики
   - Нажмите "Начать путь"

2. **Ежедневная практика:**
   - Каждый день открывается список практик
   - Изучите описание практики (нажмите стрелку для раскрытия)
   - Выполните практику
   - Отметьте галочкой выполненные практики

3. **Прогресс:**
   - Следующий день откроется автоматически после завершения всех практик текущего дня
   - Вы всегда можете вернуться к предыдущим дням для повторения
   - Прогресс сохраняется в браузере (localStorage)

## Особенности реализации

### Парсинг Markdown
Программа медитаций хранится в формате Markdown (`meditation-program.md`) и парсится на лету, что позволяет легко редактировать содержание без изменения кода.

### Система анимаций
5 различных анимаций выбираются случайным образом при отметке практики:
- **Confetti** — конфетти разлетается радиально
- **Ripple** — концентрические волны
- **Sparkle** — вращающиеся звездочки
- **Pulse** — пульсирующее свечение
- **Check Bounce** — галочка с эффектом отскока

### Адаптивный дизайн
- Mobile-first подход
- Breakpoints для планшетов и десктопов
- Оптимизированная типографика для всех экранов

## Настройка программы медитаций

Чтобы изменить программу, отредактируйте файл `public/meditation-program.md`. Формат:

```markdown
## День 1: Название дня

**Цель дня:** описание цели

### Практика 1: Название практики (5 минут)

**Что делать:**
Описание инструкций

**На чём фокусироваться:**
Описание фокуса

**На чём НЕ фокусироваться:**
Описание того, чего избегать
```
